# Scientific_Calculator
A simple yet powerful **console-based Scientific Calculator** built using C++.   This project performs basic arithmetic as well as some advanced mathematical operations.
##  Features

This calculator supports the following operations:

-  Addition
-  Subtraction
-  Multiplication
-  Division (with zero-check protection)
- % Modulus
-  Factorial
-  Square Root
-  Power (xʸ)

---

##  Concepts Used

This project is ideal for beginners and demonstrates:

- Variables and Data Types
- Conditional Statements (if-else)
- Switch Case
- Loops (do-while)
- Functions
- Math Library (`cmath`)
- Basic error handling
